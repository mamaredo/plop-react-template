# plop-react-template

### CLIで作成できるTemplate

```shell
src
|
+-- components
+---- Elements
+------ Component
+-------- Component.tsx
+-------- index.ts
+---- index.ts
+---- Head
+------ Head.tsx
+------ index.ts
|
+-- features
+---- awesome-feature
+------ api
+-------- getAwesomeFeature.ts
+------ components
+-------- AwesomeFeature.tsx
+------ types
+-------- index.ts
+------ index.ts
```

#### 手順

1. install

```shell
$ npm i
```

2. run

```
$ npm run plop
```
